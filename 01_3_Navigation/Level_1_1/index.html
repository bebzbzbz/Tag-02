Navigation: Level 1.1

Lernziel: Navigation unter Verwendung von nav und a Tags erstellen.

Eine Navigation gehört zu (fast) jeder Webseite. Mal ist sie ganz einfach aufgebaut, mal über unterschiedliche Ebenen (dazu kommen wir zu einem späteren Zeitpunkt). Du beginnst mit einer recht überschaulichen Navigation. Nutze dafür das HTML-Tag < nav >.

Darüberhinaus soll zu Beginn ein Logo positioniert werden, dass ebenfalls auf die Home-Seite verlinkt. Du kannst dafür folgenden Link nutzen, so wird automatisch ein Bild in der Größe 50px x 50px generiert.
